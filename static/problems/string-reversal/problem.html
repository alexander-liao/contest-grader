Sample Problem: String Reversal!

Given a single string of ASCII characters in the range [32 .. 127], output the string in reverse.

Input Specifications

For all points, the length of the string is less than 1024 bytes.

Output Specifications

The input will consist of a single line of input.

Subtasks

The output will consist of a single line, the input reversed.

Sample Input

cat

Sample Output

tac

Sample Input

hello, world

Sample Output

dlrow ,olleh

Sample Input

this is a string

Sample Output

gnirts a si siht
