Sample Problem: cat

Given an arbitrary about of EOF-terminated input, output that exact input.

Input Specifications

For all points, 1 ≤ N ≤ 1024 where N is the length of the input.

Output Specifications

Input will consist of an arbitrary amount of input.

Subtasks

Output will consist of the exact input.

Sample Input

hello

Sample Output

hello

Sample Input

hello
world

Sample Output

hello
world
