Sample Problem: Add One!

Given a single non-negative integer, add one to it and output it.

Input Specifications

For all points, 1 ≤ N + 1 < 232 - 1

Output Specifications

The input will consist of a single integer N.

Subtasks

The output will consist of a single integer N + 1

Sample Input

15

Sample Output

16

Sample Input

100

Sample Output

101

Sample Input

466

Sample Output

467

Sample Input

352456

Sample Output

352457

Sample Input

99

Sample Output

100
